# WW_codingExercise
QE challenge for WW

## Prerequisites:

* Selenium:

```pip install selenium```

* Webriver manager:

```pip install webdriver-manager```


## How to execute the tests?

```bash
git clone https://github.com/lavanyapothineni/WW_codingExercise.git
cd WW_codingExercise/
python Test.py
```
